# 日记评论技能

## 概述

为子楠的每日碎碎念生成 AI 个性化评论，包含情绪共情和温柔鼓励两条，总字数控制在 300~500 字之间。

## 触发场景

- 子楠写了日记，需要生成评论
- 评论字数超标，需要精简
- 查看/修改日记评论脚本

## 评论生成（diary_comment.py）

调用 LongCat API 生成两条评论：

**第一条（共情）**：精准捕捉日记核心情绪，自然融入评语，不单独罗列情绪标签，充分共情负面情绪。

**第二条（鼓励）**：给予微小、松弛的正向暗示，不给用户压力，重在接纳与陪伴。

**Prompt 要求**：
1. 第一条精准捕捉日记核心情绪，不空洞安慰
2. 充分共情，不否定、不强行开导
3. 结合日记具体事件回应
4. 第二条给予松弛的正向暗示
5. 整体语气安静治愈，像深夜独处时的自我对话
6. 不输出建议类指令（如"你明天要加油"）
7. 两条内容不能重复，各有侧重
8. **两条评论合并后总字数控制在 300~500 字之间**，精简表达

## 字数精简（condense_comment.py）

当两条评论总字数超过 500 字时，调用 AI 重新精简，不硬截断。

**用法**：
```bash
python3 scripts/condense_comment.py "第一条评论" "第二条评论"
```

**逻辑**：
- 最多重试 3 次，每次调用 AI 重新生成精简版本
- 保持情感完整，语义不断句
- 最终返回两条独立评论

## 文件位置

所有脚本统一在 `skills/diary-comment/` 目录下：

```
skills/diary-comment/
├── SKILL.md              # 技能说明文档
├── diary_comment.py      # 主脚本：生成+上传评论
└── scripts/
    └── condense_comment.py  # 字数精简核心逻辑
```

**执行命令**：
```bash
python3 skills/diary-comment/diary_comment.py
```
